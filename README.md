# Trung Le Huu Portfolio V2

Portfolio website built with Next.js App Router and TypeScript.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Validation

```bash
npm run lint
npm run build
```

## Key Files

- src/data/portfolio.ts: single source for profile, projects, awards, and skills.
- src/app/page.tsx: one-page portfolio layout and section rendering.
- src/app/globals.css: color system, typography, responsive helpers, and animation.
